TFBS Hits Summary Table

This table contains the information on where the TFBS prediction is found, along with individual TFBS hit locations.

The table is preceded by a header section listing the TF name, JASPAR ID, structural class and family, taxonomic group, information content and GC content of the TFBS profile.

Sequence ID(s)

The Sequence ID for which the TFBS is predicted.

Start

The start position of the predicted TFBS.

End

The end position of the predicted TFBS.

Strand

The strand on which the predicted TFBS is located.

Abs. Score

The absolute score of the predicted binding site given the weight matrix profile.

Rel. Score

The relative score of the predicted binding site given the weight matrix profile.

TFBS sequence

The predicted binding site sequence. Note that if the TFBS strand is negative, the sequence displayed is based on the antisense strand.
