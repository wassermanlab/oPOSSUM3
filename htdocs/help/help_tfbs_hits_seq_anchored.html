TFBS Hits Summary Table

This table contains the information on where the TFBS predictions are found, along with individual TFBS hit locations.

The table is preceded by a header section listing the TF name, JASPAR ID, structural class and family, taxonomic group, information content and GC content of the TFBS profile for the anchoring TF and the anchored (paired) TF.

Sequence ID

The Sequence ID for which the TFBS is predicted.

Anchoring TF

The Anchoring TF name.

Start

The start position of the predicted anchoring TFBS.

End

The end position of the predicted anchoring TFBS.

Strand

The chromosome strand on which the predicted anchoring TFBS is located.

Score

The absolute score for the sequence given the anchoring TFBS weight matrix profile.

% Score

The relative score for the sequence given the anchoring TFBS weight matrix profile.

Sequence

The sequence matching the given anchoring TFBS profile above the threshold match score.

Anchored TF

The Anchored TF name (paired with the anchoring TF).

Start

The start position of the predicted anchored TFBS.

End

The end position of the predicted anchored TFBS.

Strand

The chromosome strand on which the predicted anchored TFBS is located.

Score

The absolute score for the sequence given the anchored TFBS weight matrix profile.

% Score

The relative score for the sequence given the anchored TFBS weight matrix profile.

Sequence

The sequence matching the given anchored TFBS profile above the threshold match score.

Distance

The distance between the anchoring TFBS and the anchored TFBS.
